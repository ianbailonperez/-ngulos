Aplicación práctica: Cálculo de un ángulo con una escalera

Imagina una escalera apoyada sobre una pared del lado derecho. La base de la escalera se mantiene fija en el suelo (lado izquierdo). Al mover el control, la escalera gira en sentido antihorario, formando un ángulo entre el suelo y la pared.

Ángulo: 0° | Altura: 0 m

Esta animación muestra cómo se forma un ángulo entre el suelo, la escalera y la pared. La altura alcanzada por la escalera depende del ángulo de inclinación, de acuerdo con la relación trigonométrica: sen(θ) = altura / longitud. Usa el control para cambiar el ángulo y observa cómo varía la altura.

¿Cómo se calcula el ángulo? — Explicación paso a paso

En esta simulación utilizamos una escalera de longitud fija (por ejemplo 10 m) apoyada en el suelo y en una pared. El ángulo que forma la escalera con el suelo (θ) se relaciona con la altura a la que llega la escalera en la pared mediante la función trigonométrica seno.

  1. Definir los elementos:
    • longitud = longitud de la escalera (lado hipotenusa en el triángulo rectángulo formado).
    • altura = distancia vertical desde el suelo hasta el punto donde la escalera toca la pared (lado opuesto).
    • θ = ángulo entre la escalera y el suelo (lo que queremos calcular).
  2. Relación trigonométrica:

    En el triángulo rectángulo formado por suelo–escalera–pared se cumple:

    sin(θ) = altura / longitud

  3. Calcular la altura al mover el control (cuando θ es conocido):

    Si conocemos el ángulo θ (por ejemplo con el deslizador), la altura se obtiene con:

    altura = longitud × sin(θ)

    Ejemplo: si longitud = 10 m y θ = 30°, entonces altura = 10 × sin(30°) = 10 × 0.5 = 5 m.

  4. Calcular el ángulo si conocemos la altura (caso inverso):

    Si conocemos la altura y la longitud, el ángulo se calcula con la función inversa del seno (arcsin):

    θ = arcsin( altura / longitud )

    Ejemplo: si altura = 6 m y longitud = 10 m, entonces θ = arcsin(0.6) ≈ 36.87°.

  5. Restricciones y cuidados:
    • Debemos asegurarnos de que altura ≤ longitud. Si la altura es mayor que la longitud, la razón altura/longitud excede 1 y arcsin no está definida (no hay solución real).
    • En la simulación, el deslizador limita el ángulo a valores razonables (por ejemplo 0°–75°) para evitar situaciones físicamente imposibles.
  6. Relación con la animación (resumen técnico breve):
    • El programa toma el valor del deslizador (ángulo en grados).
    • Convierte grados → radianes: radians = angle * Math.PI / 180.
    • Calcula altura: height = length * Math.sin(radians) y la muestra en metros.
    • Aplica la rotación visual a la escalera con CSS: transform: rotate(-angledeg) (pregunta del signo: usamos el signo que orienta la escalera en el mismo sentido visual que la escena).

Pequeño ejemplo de código (JavaScript)

// longitud en metros
const length = 10;
// ángulo en grados (valor del deslizador)
const angleDeg = 30;
// pasar a radianes
const rad = angleDeg * Math.PI / 180;
// altura (m)
const height = length * Math.sin(rad); // -> 5 m para 30°

Con esta explicación los estudiantes pueden ver claramente la conexión entre la construcción geométrica (escalera, suelo, pared), la función trigonométrica y el cálculo numérico. Si quieres, puedo añadir un pequeño ejercicio al final donde el alumno deba introducir la altura y calcular el ángulo (con comprobación automática).
